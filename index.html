A Practical Guide to 20 Modern Excel Formulas

Master Excel's powerful new functions with step-by-step examples and clear data walkthroughs.

1. XLOOKUP

Searches a range for a match and returns the corresponding item from a second range.

AB
1IDEmployee
2102Jane
3105Mike
4108Sue

Example:=XLOOKUP(105, A2:A4, B2:B4)

Show Walkthrough

Walkthrough: Excel searches for the ID 105 in the ID column (A2:A4) and returns the value from the same position in the Employee column (B2:B4).

Result in cell C2:

C
2Mike

2. FILTER

Filters a range of data based on criteria, returning all matching records.

AB
1RegionSales
2North2500
3South1800
4North3200

Example:=FILTER(A2:B4, A2:A4="North")

Show Walkthrough

Walkthrough: The formula checks the Region column for "North" and spills a new table containing only the two matching rows and their sales data.

Spilled Result:

CD
2North2500
3North3200

3. SORT

Sorts the contents of a range or array.

AB
1ProductPrice
2Banana$1.20
3Apple$1.50
4Cherry$2.00

Example:=SORT(A2:B4, 2, -1)

Show Walkthrough

Walkthrough: It sorts the table based on the 2nd column (Price) in descending order (-1), returning a new, sorted table.

Spilled Result (Sorted by Price Desc):

CD
2Cherry$2.00
3Apple$1.50
4Banana$1.20

Assigns names to calculation results to improve readability.

A

Show Walkthrough

Walkthrough: It assigns the name "rate" to the value 0.05, then multiplies the value in A2 by "rate".

Assembles custom, reusable functions without VBA.

A

Show Walkthrough
